### Installing
  Using VSCODE:
    - Download project files
    - Ensure Docker is running, run:
      `docker build -t stock_prediction_app .`
      `docker run -detatch stock_prediction_app`
    - Then go to front end, make sure npm installed, run:
      `npm run dev`
